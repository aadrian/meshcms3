MeshCMS provides a library of custom tags to ease the creation of themes. Some are mandatory, others can be useful in many situations. The following sections explain each tag separately. As an example, the <cms:tagname> format is used, which implies the following declaration at the beginning of the theme file:

<%@ taglib uri="meshcms-taglib" prefix="cms" %>

Mandatory tags are: <cms:editor>, <cms:pagehead> and <cms:pagebody>.
