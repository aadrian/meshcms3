Welcome to MeshCMS. This is a sample home page that you can easily edit after you have logged in.

Here is a list of tasks you can try to test this CMS:

  • click on Login and insert username admin and password admin;

  • when you see the Control Panel, click on Site Home to go back to this page, then click on Edit this page to open the WYSIWYG editor;

  • experiment with the editor and then save the page;

  • go to Manage Pages in the Control Panel and create a new page under the home page, then edit it;

  • go back to Manage Pages and try changing themes (pages whose theme field is empty will inherit the theme from the parent)

You need to allow popups from the site where MeshCMS is running, otherwise you won't be able to use the file browser to choose links and images while editing pages.

Note: all themes except Flowers are rough adaptations of GPL Mambo themes downloaded from Mambo Hut.
