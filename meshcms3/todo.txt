MeshCMS 3.0 Release (~ 10-Jul-2006)
-----------------------------------

#1 - tool to edit file dates in the file manager
#2 - tag to load the ALIB script (for menus)
#3 - new default admin theme
#4 - documentation (at least for the new multi-site support)
#5 - www.meshcms.org (site + theme + hosting + required applications to install)
#6 - update translations
#7 - new module: headlines
#8 - new module: search
#9 - new module: comments
#10 - new module support with multi-parameters (+ XUL)
#11 - community themes to choose from
x #12 - build process improvements (support for versions: alpha, beta, RC; new distribution packaging)
#13 - add UnitTests for the core functions (No WebUnit Tests)
#14 - automatic web tests with Selenium (http://www.openqa.org/selenium/)  or Sahi (http://sahi.sourceforge.net/)
#15 - add better logging with Log4J
#16 - some loading and performance tests of the multi-site feature with JMeter(http://jakarta.apache.org/jmeter/index.html)

#17 - IMPORTANT: a better and clear decision and planninig for the so wanted "static export" feature.

x #18 - add Checkstyle (http://checkstyle.sourceforge.net/), to automaticaly help cleanup the code.
x #19 - Sandbox area for all developers in the Subversion repository (to have a better place for experimenting).
#20 - MOST IMPORTANT: find out how normal webapplications (that worked in their virtual host/doamin) will work now with this multi-site support.
#21 - update the JavaDocs for the entire code base.
#22 - architecture diagram and documentation of the CMS
#23 - code refactorings, to make it easier to understand for newbies
#24 - new and better and table-less theme for the default site (something that is more reusable than the "flowers" theme).
x #25 - add support for "Tag Library Documentation Generator"
x #26 - tune checkstyles config files, to obtain better results.
#27 - try to cover with unit tests as much as possible from the actual code base
