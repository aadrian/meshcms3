MeshCMS can host a virtually unlimited number of sites in a single installation. If you look at the Site Manager, you see that two sample sites are provided here. To see them you must tell your PC to map the domains (www.)virtual.com and (www.)static.com to the IP of the machine that is running MeshCMS. For example, on Windows you should run this command:

%SystemRoot%\system32\notepad.exe %SystemRoot%\system32\drivers\etc\hosts

and add these lines to that file:

127.0.0.1       www.virtual.com
127.0.0.1       www.static.com

Then you can visit the virtual sites. Please note that in a testing environment you must keep your port number and context path in the URLs. For example, if you are viewing this page at http://localhost:8080/samples/virtualsites, the URLs for the virtual sites will be http://www.virtual.com:8080/samples and http://www.static.com:8080/samples.
